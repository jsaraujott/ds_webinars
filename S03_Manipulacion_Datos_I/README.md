En este caso de estudio y ejercicios se busca prácticar los fundamentos de manipulación de datos con **Pandas**. Adicionalmente, se va a introducir los primeros criterios para un adecuado ordenamiento de proyectos de análisis de datos:

* Entendimiento del contexto
* Entendimiento de los datos
* Preparación de los datos
* Análisis de datos
